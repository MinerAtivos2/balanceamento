Sobre o Projeto

Rebalanceamento de portfólio

🎯 O Projeto

Plataforma para análise e rebalanceamento de portfólios de investimentos na B3 (Bolsa de Valores Brasileira). Oferece ferramentas para otimização de alocação de ativos usando a Teoria Moderna de Portfólio (Markowitz) e avaliação via Método Barsi/Bazin.

⚙️ Funcionalidades

  • Dashboard: Visão geral com gráficos interativos
  • Gestão de Posições: Adicione, edite e remova ativos do portfólio
  • Preço-Teto (Barsi): Identifica oportunidades de compra baseado em dividendos
  • Rebalanceamento: Otimização de alocação baseada em volatilidade
  • Dados Atualizados: Histórico diário via yfinance para análises

🛠️ Tecnologias

HTML5 / CSS3 JavaScript (ES6+) Chart.js
⚠️ Aviso Legal: Esta plataforma é fornecida como ferramenta educacional. Não constitui recomendação de investimento. Consulte um profissional qualificado.
