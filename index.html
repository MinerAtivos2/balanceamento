Sobre o B3 Rebalanceamento & IA

O Projeto

Uma plataforma de código aberto para análise e rebalanceamento de portfólios de investimentos na B3 (Bolsa de Valores Brasileira). Desenvolvida em Python com interface web, oferece ferramentas para otimização de alocação de ativos usando a Teoria Moderna de Portfólio (Markowitz).

Funcionalidades Principais

  • Dashboard de Portfólio: Visualize suas posições e rentabilidade em tempo real
  • Análise de Preço-Teto: Método Barsi/Bazin para identificar boas oportunidades de compra
  • Rebalanceamento Markowitz: Otimização de alocação para máximo Sharpe Ratio
  • Análise de Volatilidade: Avalie o risco do seu portfólio
  • Automação via GitHub Actions: Atualizações automáticas de dados diariamente

Tecnologias Utilizadas

  • Backend: Python, yfinance, pandas, scipy
  • Frontend: HTML5, CSS3, JavaScript vanilla
  • Automação: GitHub Actions, GitHub Pages
  • Hospedagem: GitHub Pages (estático)

Como Usar

  1. Clone o repositório do GitHub
  2. Instale as dependências Python: pip install -r requirements.txt
  3. Execute o script de coleta de dados: python scripts/data_fetcher.py
  4. Execute as análises: python scripts/portfolio_analyzer.py
  5. Abra web/index.html no navegador para visualizar os resultados
