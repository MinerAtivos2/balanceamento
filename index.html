Sobre o Projeto

B3 Rebalanceamento & IA

🎯 O Projeto

Plataforma de código aberto para análise e rebalanceamento de portfólios de investimentos na B3 (Bolsa de Valores Brasileira). Oferece ferramentas para otimização de alocação de ativos usando a Teoria Moderna de Portfólio (Markowitz) e avaliação via Método Barsi/Bazin.

⚙️ Funcionalidades

  • Dashboard: Visão geral com gráficos interativos
  • Gestão de Posições: Adicione, edite e remova ativos do portfólio
  • Preço-Teto (Barsi): Identifica oportunidades de compra baseado em dividendos
  • Rebalanceamento (Markowitz): Otimização para máximo Sharpe Ratio
  • Dados em Tempo Real: Integração com APIs públicas para dados atualizados

🛠️ Tecnologias

HTML5 / CSS3 JavaScript (ES6+) Chart.js Brapi API GitHub Pages

⚙️ Configurações

Para buscar dados em tempo real, você pode utilizar um token da API Brapi.dev (opcional).

⚠️ Isso atualizará o token para todos os usuários no banco de dados.

⚠️ Aviso Legal: Este projeto é fornecido como ferramenta educacional. Não constitui recomendação de investimento. Consulte um profissional qualificado.
